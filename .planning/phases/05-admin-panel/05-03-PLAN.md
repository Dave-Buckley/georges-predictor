**Server actions:**

1. `getCloseGameweekSummary(gameweekId: string)` — NOT a form action, just an async function (called by the dialog to fetch data).
   - requireAdmin() guard
   - Query fixtures for this GW: count total, count by status (FINISHED, CANCELLED, POSTPONED vs other)
   - Blocking fixtures = those NOT in [FINISHED, CANCELLED, POSTPONED] — return their names (home vs away) and status
   - Query bonus_awards where gameweek_id matches and awarded IS NULL — count pending
   - Query bonus_schedule for this GW — is bonus confirmed?
   - Query prediction_scores for this GW — count total points distributed, count of scored fixtures
   - Return typed summary object:
     ```typescript
     interface CloseGameweekSummary {
       totalFixtures: number
       finishedFixtures: number
       blockingFixtures: Array<{ id: string; label: string; status: string }>
       pendingBonusAwards: number
       bonusConfirmed: boolean
       totalPointsDistributed: number
       canClose: boolean
     }
     ```

2. `closeGameweek(formData: FormData)` — validates with closeGameweekSchema.
   - requireAdmin() guard
   - Re-check canClose (don't trust client state — re-query blocking fixtures and pending awards)
   - If canClose is false, return { error: 'Cannot close: ...' } with specific reason
   - UPDATE gameweeks SET closed_at = now(), closed_by = auth.userId WHERE id = gameweek_id
   - INSERT admin_notification (type: 'gw_complete', title: 'Gameweek N closed', message: 'Closed by admin')
   - revalidatePath('/admin', 'layout') — revalidate dashboard and all admin pages
   - Return { success: true }

3. `reopenGameweek(formData: FormData)` — validates with reopenGameweekSchema.
   - requireAdmin() guard
   - UPDATE gameweeks SET closed_at = null, closed_by = null WHERE id = gameweek_id
   - INSERT admin_notification (type: 'system', title: 'Gameweek N reopened')
   - revalidatePath('/admin', 'layout')
   - Return { success: true }

4. `updateAdminSettings(formData: FormData)` — for email notification toggles.
   - requireAdmin() guard
   - Parse boolean toggles from FormData: email_bonus_reminders, email_gw_complete, email_prize_triggered
   - UPSERT admin_settings row (onConflict: 'admin_user_id') with the toggle values and updated_at = now()
   - revalidatePath('/admin/settings')
   - Return { success: true }

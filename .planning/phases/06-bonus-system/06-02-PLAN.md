<objective>
Member-facing bonus pick — integrated into prediction submission flow.

Purpose: Members can see the active bonus for a gameweek, tap a star on a fixture card to select their bonus fixture, and submit the pick alongside their predictions. The bonus pick is mandatory when a bonus is active, uses the same lockout rules as predictions, and is saved as a bonus_awards row via member session client (RLS enforced).

Output: Extended submitPredictions action, bonus pick UI in fixture cards, bonus banner component, gameweek page data fetching for bonus data.

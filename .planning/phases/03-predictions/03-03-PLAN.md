**Create `src/components/predictions/predictions-table.tsx`:**

A client component ('use client') rendering a scrollable grid table:
- Rows = approved members (sorted alphabetically by display_name)
- Columns = fixtures in the selected gameweek (sorted by kickoff_time)
- Column headers: team abbreviations (TLA) e.g. "ARS v CHE" or just home TLA + away TLA stacked
- Each cell shows the member's prediction as "H-A" (e.g. "2-1") or "—" if no prediction
- Highlight cells: green background if prediction matches the actual result (if result exists), for visual feedback to George

Props interface:
```typescript
interface PredictionsTableProps {
  members: Array<{ id: string; display_name: string }>
  fixtures: FixtureWithTeams[]
  predictions: Array<{
    member_id: string
    fixture_id: string
    home_score: number
    away_score: number
  }>
}
```

**Table design:**
- First column (sticky left): member display_name
- Remaining columns: one per fixture, header shows home TLA vs away TLA
- Table container: `overflow-x-auto` for horizontal scroll on mobile
- Compact cells: `text-xs` or `text-sm`, minimal padding
- Alternating row backgrounds for readability
- "Submitted" indicator: members with at least one prediction get a small green dot next to their name; members with zero predictions get a red dot
- If no members have submitted, show an empty state message

**Styling:** Admin pages use white/gray theme (not dark slate member theme). Match the existing admin card styling: `bg-white border border-gray-200 rounded-2xl`.

2. **Data fetching (all via `createAdminClient()`):**
```typescript
import { createAdminClient } from '@/lib/supabase/admin'

const supabaseAdmin = createAdminClient()

// All approved members
const { data: members } = await supabaseAdmin
  .from('members')
  .select('id, display_name')
  .eq('approval_status', 'approved')
  .order('display_name')

// Gameweek + fixtures
const { data: gameweek } = await supabaseAdmin
  .from('gameweeks')
  .select('*')
  .eq('number', selectedGw)
  .single()

const { data: fixtures } = await supabaseAdmin
  .from('fixtures')
  .select('*, home_team:teams!home_team_id(*), away_team:teams!away_team_id(*), gameweek:gameweeks!gameweek_id(*)')
  .eq('gameweek_id', gameweek.id)
  .order('kickoff_time')

// ALL predictions for this gameweek's fixtures (admin sees everything)
const fixtureIds = fixtures.map(f => f.id)
const { data: predictions } = await supabaseAdmin
  .from('predictions')
  .select('member_id, fixture_id, home_score, away_score')
  .in('fixture_id', fixtureIds)
```

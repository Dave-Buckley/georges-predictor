- BST/GMT timezone shown explicitly on EVERY kick-off time (e.g., "15:00 BST")

- Rescheduled fixtures show a visible "Rescheduled" badge

- Navigation via prev/next arrows AND a dropdown picker

- Gameweeks marked "Complete" with a green badge

- Pulsing red "LIVE" badge on in-play fixtures (no live score — Phase 4)

- All-fixtures page accessible to logged-in members and admins (not public)

    - `FINISHED`: Show final score prominently. "FT" badge. Full greyed state.
    - `POSTPONED`/`SUSPENDED`: Orange "Postponed" badge, greyed text, no time shown
    - `CANCELLED`/`AWARDED`: Show status badge, greyed out

    **Countdown implementation:** Only render countdown when `showCountdown` is true AND fixture kicks off today. Use `isToday()` from timezone.ts to determine. The countdown is purely cosmetic — server-side lockout (Phase 3) is the real enforcement.

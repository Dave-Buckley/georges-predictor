Admin Guide

George's Predictor — running the league week to week
https://georges-predictor.vercel.app/admin

Everything you need to run the season. Each week takes about 10 minutes of your time if you stick to the routine in section 1. Fixtures, scores and league-table updates all happen automatically — you just step in for the things only a human can decide: approvals, bonuses, prizes, and the end-of-week close.

1. Weekly routine

What to do each matchweek

  1. Wednesday before gameweek — set the bonus for the upcoming week (Admin → Gameweeks → pick the gameweek → Set bonus).
  2. Friday morning — site auto-syncs fixtures from football-data.org. Spot-check the latest sync time on the admin dashboard.
  3. After every match finishes — scores come in automatically. Check a handful against the BBC live scores to catch any data glitches. You can manually edit a score from Admin → Gameweeks → the gameweek → tap a fixture.
  4. After the final whistle of the gameweek — go to Admin → Bonuses, confirm each member's bonus award (thumbs-up/down for each). Then go to Admin → Dashboard and click Close gameweek. This fires the weekly email report to everyone and locks the week.
  5. Anytime — approve new signups as they come in. You'll get an email notification. Admin → Members → Pending tab → Approve.
Rule of thumb: don't close the gameweek until you've (a) confirmed all bonus awards and (b) spot-checked at least 3 match scores against a live source. Reopening a closed gameweek is possible but undoes the email report.

2. Approving new members

When someone signs up you (and Dave) get an email. Click the link, or go to Admin → Members and open the Pending tab.

  1. Check the display name — does it match someone from last year?
  2. If yes: approve. Their carry-over points stay attached.
  3. If they typed a brand-new name when they should have picked from the list: don't approve yet. Message them on WhatsApp to confirm. If they're actually an existing member, you can fix it by editing their display name in Admin → Members → edit (type their real league name) and updating their starting points.
  4. If they're a genuine newbie: approve with 0 starting points. They're in.
  5. Approving sends them a magic login link automatically. They should get it within a minute — tell them to check spam if not.
Reject only if you don't know them. Rejected emails are blocked from re-registering under the same address.

3. Setting the weekly bonus

Every gameweek has one active bonus. Members apply it to one fixture of their choice. Set it before the first kickoff of the gameweek or members can't pick.

  1. Admin → Gameweeks → pick the upcoming gameweek.
  2. Click Set bonus.
  3. Pick one of the types from the dropdown. Scroll to see them all.
  4. Confirm. Members can now choose their fixture.

Bonus types (from the dropdown):

NameWhat it rewards
Double BubbleExact score on bonus pick — points double.
Clean SheetPick a team; bonus if they concede 0.
Goals Galore3+ goals in the match.
Golden GloryExact score on bonus pick — 60 pts.
BankerSafer: correct result on bonus pick.

4. Scoring & manual overrides

Scores sync from football-data.org. 99% of the time they're right. Occasionally a match ends with a VAR decision that shifts a score, or a data provider lag leaves a result briefly wrong.

Check every week. When Friday's fixture sync runs, eyeball a few past-week results against the BBC or Sky live scores. If you spot a mismatch:

  1. Admin → Gameweeks → pick the affected gameweek.
  2. Tap the fixture with the wrong score.
  3. Click Edit, enter the correct score, save.
  4. The system automatically recalculates every member's points for that fixture.
You can edit scores even after the gameweek is closed. Recalculation is automatic. Members' points update on the standings page within a minute.

5. Closing a gameweek

Closing finalises the week and triggers the weekly email report to all members. Do this once all matches have finished and all bonus awards are confirmed.

  1. Admin → Dashboard. If all fixtures have finished, you'll see a green "GW ready to close" card.
  2. Click Close gameweek.
  3. The dialog shows you a summary: fixtures played, bonus awards granted, total points distributed. Confirm.
  4. Weekly emails go out automatically to everyone who opted in.
If a match is rescheduled mid-week: the sync will move it automatically. If it can't finish during this gameweek's window, the site will tell you there's a blocker when you try to close — the match gets marked postponed and dropped from scoring, which unblocks the close.

6. Prizes

Prizes are one-off awards that trigger on specific moments (e.g. "first to hit 100 pts", "best gameweek of the month"). Admin → Prizes shows any triggered prizes waiting for your confirmation.

  1. Review each triggered prize with the snapshot it was calculated from.
  2. Approve to announce, or reject if the data looks wrong.
  3. Approved prizes show up on the member's profile page and in the next weekly email.

7. Managing members

Admin → Members lets you edit anyone's row:

  • Display name — how they appear on the league table.
  • Email — where reminders and magic links go.
  • Starting points — their carry-over score. Change this if you ever need to fix an import error.
  • Approval status — Approved / Pending / Rejected.

Members can edit their own name, email and email-reminder preference from their Profile page. But you can override anything they've set.

8. Pre-season picks

Each season opens with pre-season predictions (top 4, 10th, bottom 3, promoted teams). Members submit via their Pre-Season page before GW1. Scored at season end.

Admin → Pre-Season lets you:

  • See who's submitted and who hasn't.
  • Enter the actual standings once the season finishes.
  • Confirm each member's earned awards.

9. End of season

When the final gameweek closes, Admin → Season Rollover runs a step-by-step wizard: archive this season, enter actuals, confirm pre-season awards, launch next season. Takes about 15 minutes.

10. Things to watch out for

"Failed to send magic link" on approval

If the member's email already exists in Supabase (rare — happens if you delete+readd), the system falls back to a regular email. Message them directly if unsure, or ask Dave to reset their password from the backend.

A match score looks wrong after sync

Don't wait — fix it manually (section 4). Members get confused fast when their points jump around.

Nobody can log in

Check the admin dashboard's Fixture Sync card. If the last sync is more than 2 days old, the service may have paused — redeploy or contact Dave.

"I didn't get my weekly email"

Most likely spam. If you've closed the gameweek and they still don't have it, check Admin → Recent notifications for any "report send failed" entries.
